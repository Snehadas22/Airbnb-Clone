# Wanderlust 🏡
A full-stack property rental and listing platform inspired by Airbnb, built with modern web technologies. Users can explore properties, create listings, upload images, manage bookings, and securely authenticate their accounts.

## 🚀 Project Highlights

- Property listing and management system
- Secure user authentication and authorization
- Image upload and cloud storage integration
- Interactive property details and reviews
- Responsive and user-friendly interface
- RESTful API architecture
- Complete CRUD operations
